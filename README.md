# Caderno do Hortifruti

## Sobre

O **Caderno do Hortifruti** é uma ferramenta digital criada para facilitar a consulta dos operadores de caixa da **Casa de Carnes Brasil**. 

Esse caderno foi desenvolvido para oferecer uma forma rápida e eficiente de:
- Consultar códigos de barras de produtos de frutas e hortaliças
- Visualizar informações detalhadas de cada produto
- Imprimir o caderno em formato A4 para referência física no caixa

## Características

- **Interface intuitiva**: Design limpo e fácil de usar
- **Códigos de barras**: Cada produto possui um código de barras escaneável
- **Edição de produtos**: Modo de edição para adicionar, remover ou modificar produtos
- **Impressão otimizada**: Layout otimizado para impressão em papel A4
- **Armazenamento local**: Os dados são salvos automaticamente no navegador

## Como usar

1. Abra o arquivo `index.html` em um navegador web
2. Navegue pela lista de produtos de frutas e hortaliças
3. Para editar: clique em "Editar Caderno"
4. Para imprimir: clique em "Imprimir"

## Operadores de Caixa

Para os operadores de caixa da Casa de Carnes Brasil: utilize este caderno para consultar rapidamente os códigos de produtos durante o atendimento. O layout foi otimizado para impressão, permitindo uma cópia física sempre à mão no ponto de venda.
